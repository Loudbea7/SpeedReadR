## Requirements
- beautifulsoup4
- EbookLib
- Kivy
- kivymd
- Markdown
- pypdf
- sqlmodel
- striprtf
- wakepy


## RUN
After installing all requirements run

```
python speed_read_r.py
```

It is recommended to use `pyenv` and `poetry` to manage dependencies versions independently of you OS.
